Responsive

SAMSUNG

This is a screenshot of what Samsung's website looks like for mobile view. I really like how responsive it was and how things shrunk down appropriately. I have tried to add the same thing to my website so that things would adjust well for any size.

Contrast

CHASE

This is Chase the bank's website and I like how there is largely just two colors, white and blue. The contrast helps keep a very clean look and familiar feel to it. I tried to apply the same type of concept with this page by using black and white which have pretty good contrast.

White Space

TIME

I liked the white space used here in the Time's website. This really helps make the website have plenty of room to breath. In this website I do simmilar things to make sure that there is plenty of space on the sides so that the page doesn't feel cramped.
